# Skill: Odoo Marketplace Management

ชุดคำสั่งและขั้นตอนการทำงาน (Workflows) สำหรับจัดการ Odoo Multi-Vendor Marketplace ผ่าน Odoo MCP

## 1. การสำรวจร้านค้า (Seller Exploration)
- **ตรวจสอบรายชื่อร้านค้า:** ใช้ `odoo-mcp-tarad_odoo_search_read` บนโมเดล `seller.shop`
- **ตรวจสอบสถานะผู้ขาย:** ตรวจสอบฟิลด์ `state` (เช่น `approved`, `pending`, `new`)
- **ข้อมูลผู้ขาย (Partner):** ร้านค้าจะเชื่อมโยงกับ `res.partner` โดยมีฟิลด์ `seller: True`

## 2. การจัดการสินค้า (Product Management)
- **ค้นหาสินค้าของ Seller:** ค้นหาใน `product.template` โดยใช้ filter `marketplace_seller_id != False`
- **ตรวจสอบสถานะการอนุมัติ:** ใช้ฟิลด์ `status` ใน `product.template`
  - `draft`: รอดำเนินการ
  - `pending`: รออนุมัติ
  - `approved`: อนุมัติแล้ว (แสดงบนหน้าเว็บ)
  - `rejected`: ปฏิเสธการอนุมัติ

## 3. คำสั่งที่ใช้บ่อย (Common Commands)
- **สรุปยอดขายแยกตามร้าน:**
  `odoo-mcp-tarad_odoo_search_read(model='sale.order.line', domain=[['marketplace_seller_id', '!=', False]], fields=['price_subtotal', 'marketplace_seller_id'])`
- **ตรวจสอบการชำระเงินให้ Seller:**
  `odoo-mcp-tarad_odoo_search_read(model='seller.payment', fields=['seller_id', 'amount', 'state'])`
